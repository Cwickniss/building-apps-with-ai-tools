Building Simple Web Apps with AI Tools

Empowering you to create your own useful web applications with the help of AI – no professional developer experience required.

This course IS about:
  • Learning to be more productive with AI tools
  • Building personal, local-first web apps for your own needs
  • Using pure HTML/JavaScript (no frameworks required)
  • Leveraging AI assistance for problem-solving and code generation
  • Gaining confidence to create solutions for problems that matter to you
  • For technically curious non-developers, low-code/no-code users, and anyone who ever thought "I wish there was an app for that"
This course is NOT about:
  • Building commercial or enterprise applications
  • Deploying apps for profit
  • Complex software architecture or advanced programming
